# Patient archive

#### (**Client Only**)

Archive client patient

**URL** : `/api/client/patient/:id/archive`

**Params**
1. **id** = Patient Id

**Method** : `POST`

**Auth required** : YES

## Success Response

**Code** : `200 OK`

**Post Data**

```json
{
}
```

**Content examples**

```json
{
    "success":true,
    "message":"Patient has been deleted."
}
```
